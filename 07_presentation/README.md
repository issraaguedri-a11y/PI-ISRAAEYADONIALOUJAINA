# 07 — Présentation

Placez ici les supports de la **soutenance finale** :

- Les slides (PowerPoint, Google Slides exporté, Keynote, Beamer, etc.).
- Une éventuelle vidéo de démonstration de secours (en cas de souci réseau).
- Le script ou les notes de présentation.

## Format

- **Durée** : 20 minutes de présentation + 10 minutes de questions.
- **Démonstration live** de l'application web : 5 minutes minimum.
- Chaque membre de l'équipe doit prendre la parole.

## Structure suggérée

```
07_presentation/
├── slides.pptx          # ou .pdf, .key, etc.
├── demo_backup.mp4      # vidéo de secours (optionnel)
├── notes.md             # script de présentation
└── README.md
```
